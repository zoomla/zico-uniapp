# zico-uniapp zico在UNIAPP下的开发标准示例
基于逐浪软件(逐浪字库)旗下zico图标库集成UNIAPP开发的zico示例，代码位置：static/zico.scss


免费国产图标库 http://ico.z01.com
接口基于逐浪CMS http://code.z01.com/apidoc
